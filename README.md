# Lab Threads & AsyncTask 

##  Présentation

Ce lab Android a pour objectif de comprendre la programmation asynchrone en Java sur Android.

L’application permet :

* de lancer un traitement long avec un Thread
* d’exécuter un calcul lourd avec AsyncTask
* de garder une interface fluide pendant l’exécution
* d’afficher une barre de progression et des messages de statut

---

#  Technologies utilisées

* Java
* Android Studio
* Thread
* Handler
* AsyncTask
* ProgressBar
* ImageView

---

#  Fonctionnalités

✅ Chargement d’image en arrière-plan
✅ Calcul lourd sans bloquer l’interface
✅ Mise à jour de ProgressBar
✅ Affichage de Toast pendant le traitement
✅ Interface toujours réactive

---

#  Concepts appris

* UI Thread vs Worker Thread
* Création de Thread avec Runnable
* Utilisation de Handler et post()
* AsyncTask et progression
* Gestion des traitements longs Android

---

#  Tests réalisés

* Chargement image avec Thread
